# Coinrat - Frontend App for trading platform

See: https://github.com/achse/coinrat 

## Install
Yarn & Flow
```bash
yarn isntall
sudo yarn global upgrade flow-typed
flow-typed install
```

.env
```bash
cp .env.example .env
```
Change socket port in `.env` if needed.

## Run:
`yarn start`
